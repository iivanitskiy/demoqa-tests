## 🛠️ **Технические детали**

### **Используемые технологии:**
- **Язык:** Python 3.x
- **Фреймворк тестирования:** Pytest
- **UI автоматизация:** Selenium WebDriver
- **API тестирование:** библиотека Requests
- **Отчетность:** Allure Framework
- **Управление локаторами:** Page Object Model (POM)
- **Конфигурация:** INI-файлы + JSON для тестовых данных

### **Тестовые данные:**
- **Пользователь для логина:** `Ivanovaqa` / `Ivanovaqa1!`
- **Данные для регистрации:** сгенерированные значения из `test_data.json`
- **Тестируемые книги:** "Git Pocket Guide" (ISBN: 9781449325862), "Programming JavaScript Applications" (ISBN: 9781491950296)

### **Особенности реализации:**
1. Использование фикстур Pytest для инициализации драйвера и тестовых данных
2. Поддержка параметризованных тестов для проверки различных сценариев
3. Интеграция с Allure для детализированных отчетов
4. Чистая архитектура с разделением на слои (API, Pages, Locators, Tests)
5. Обработка алертов и ожиданий через WebDriverWait
